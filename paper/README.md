# Paper: Learning the GHZ Game with Multi-Agent Reinforcement Learning

## Overleaf Setup

1. Create a new Overleaf project with the **IEEE Conference** template
2. Upload `main.tex` as the main document
3. Create a `figures/` folder and upload all files from `../figures/`
4. Compile with **pdfLaTeX**

## Files

- `main.tex` — Complete paper source
- `../figures/fig1_game_schematic.pdf` — GHZ game schematic
- `../figures/fig2_training_dynamics.pdf` — Training curves
- `../figures/fig3_policy_evolution.pdf` — Policy evolution
- `../figures/fig4_quantum_vs_classical.pdf` — Quantum vs classical bar chart
- `../figures/fig5_quantum_interference.pdf` — Quantum interference
- `../figures/fig6_ablations.pdf` — Ablation results
- `../figures/fig7_multi_seed_summary.pdf` — Multi-seed summary
